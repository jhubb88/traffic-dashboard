Dashboard Layout

Where should the incident panel live relative to the map?

Left Sidebar

Panel on left, map fills remaining space. Classic dashboard feel. Easy to scan incidents while viewing the map.

Right Sidebar

Map on left (primary), panel on right. Natural reading flow — eyes move left to right from map to details.

Bottom Panel

Full-width map on top, incident list scrolls below. Better on mobile/widescreen. Map gets maximum width.
